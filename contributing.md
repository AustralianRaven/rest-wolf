
# Running Locally

## How to Run Locally

```bash
# install dependencies and setup
npm run setup
```

```bash
# run react app (terminal 1)
npm run dev:web

# run electron app (terminal 2)
npm run dev:electron
```

## Making an .exe

```bash
npm run build:web
# You should find the exe and other files in the `win-unpacked` dir
```
